## CheckBox Tree Components
The CheckBox Tree is implemented as a Model-View-Controller (MVC) pattern and
has three primary components plus a set of sub-components and extensions.

* [Tree](wiki/CheckBox-Tree)
* [Model](wiki/Model)
* [Store](wiki/Store)
